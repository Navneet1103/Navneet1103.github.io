# My Portfolio Website

This is my personal portfolio website that showcases my skills, projects, and achievements as a web developer. I created this website using HTML, CSS, JavaScript, and Bootstrap. You can visit my website at [https://navneet1103.github.io/].

## Installation and Usage

To run this website locally, you need to have [Git] and a web browser installed on your computer. You can follow these steps to clone and open the website:

```bash
# Clone the repository
git clone https://github.com/Navneet1103/Navneet1103.github.io.git

# Change the directory
cd portfolio-website

# Open the index.html file in your browser
```

## Technologies Used

- [HTML]: A markup language for creating web pages and web applications.
- [CSS]: A style sheet language for describing the presentation of web pages.
- [JavaScript]: A scripting language for adding interactivity and functionality to web pages.
- [Bootstrap]: A framework for developing responsive and mobile-first web sites.
